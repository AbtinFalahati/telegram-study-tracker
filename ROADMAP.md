# Roadmap

This roadmap is intentionally provisional. We will refine each sprint before implementation, based on what we learn and what the product needs.

## Sprint 0 — Foundation

- Define repository structure and documentation conventions.
- Add project metadata and Git hygiene files.
- Record initial architectural decisions.

## Sprint 1 — Bot skeleton

- Create the Python package and minimal application entry point.
- Connect safely to the Telegram Bot API using configuration from environment variables.
- Add a basic command and its automated tests where practical.
- Document local setup and how to run the bot.

## Future milestones

1. Record a study session.
2. Persist sessions in SQLite.
3. View recent sessions and simple summaries.
4. Improve validation, error handling, and test coverage.
5. Prepare contributor guidance and a portfolio-quality release.

Features are not commitments until planned as a sprint. We will keep scope small and choose the next item after reviewing the previous sprint.
